    }

    Layout:
    - Wrap in `<Card className="flex flex-col gap-2 p-4 min-h-[120px]">`.
    - Row 1: `<span className="text-[11px] uppercase tracking-wide text-muted-foreground">{label}</span>`.

    - Row 3 (flex justify-between items-end):
      - Left: if `deltaPct != null`, render `"↑ 18.0%"` / `"↓ 5.2%"` with:
        - positive → `text-emerald-600`, arrow `↑`
        - negative → `text-red-600`, arrow `↓`
        - zero → `text-muted-foreground`, arrow `·`
        All wrapped in `<span className="text-xs font-medium tabular-nums">`. Append `deltaLabel` after a space in muted text.
      - Right: if `sparklineData && sparklineData.length > 0`, render inline SVG 60×20:
        ```tsx
        <svg width={60} height={20} viewBox="0 0 60 20" aria-hidden="true">
          <path d={buildSparkPath(sparklineData)} fill="none" stroke={accentColor} strokeWidth={1.25} />
        </svg>
        ```
        where `buildSparkPath(data)` normalizes values to y range [2, 18] and x steps across [0, 60]. Include the helper at the top of the file.

    Edge cases:
    - Empty `sparklineData` → omit the SVG entirely.
    - Single data point → draw a horizontal line at its normalized y.
    - All equal values → y-center (y=10) horizontal line.
    - `deltaPct === null` or undefined → hide the delta row left side; still render label and value.

    Do NOT call `Date.now()` or `useState` — this is a pure presentational component.
