      - Primary: `Apply` — calls `onChange(localState)`, persists `localStorage.setItem(MANAGEMENT_STORAGE_KEY, JSON.stringify(localState))`, then closes drawer (`onOpenChange(false)`).
      - Secondary: `Reset to defaults` — sets local state to `DEFAULT_MANAGEMENT_SCENARIO`, calls `onChange(DEFAULT_MANAGEMENT_SCENARIO)`, and `localStorage.removeItem(MANAGEMENT_STORAGE_KEY)`.

    Also export a helper:
    ```typescript
    export function loadScenarioFromStorage(): ManagementScenario {
      if (typeof window === 'undefined') return DEFAULT_MANAGEMENT_SCENARIO;
      try {
        const raw = window.localStorage.getItem(MANAGEMENT_STORAGE_KEY);
        if (!raw) return DEFAULT_MANAGEMENT_SCENARIO;
        const parsed = JSON.parse(raw);
        return { ...DEFAULT_MANAGEMENT_SCENARIO, ...parsed };
      } catch {
        return DEFAULT_MANAGEMENT_SCENARIO;
      }
    }
    ```

    Guardrails:
    - All state transitions immutable; never mutate props.
    - Do NOT call `localStorage` at module scope (SSR safety). Only inside event handlers or `loadScenarioFromStorage`.
    - Input `onChange` coerces to number via `Number(e.target.value)`; guard `isNaN` → keep previous value.
