**Invisible offset + visible bar technique:**

The waterfall pattern requires two Bar series per data group:
1. **Invisible base bar:** `<Bar dataKey="base" stackId="stack" fill="transparent" />` — positions the visible bar
2. **Visible value bar:** `<Bar dataKey="absValue" stackId="stack">` with `<Cell fill={...} />` per bar

Data transformation (inside component):
```typescript
const chartData = waterfall.map(bar => ({
  label: bar.label,
  base: bar.base,
  absValue: Math.abs(bar.value),
  fill: bar.isTotal ? '#10B981' : '#EF4444',  // emerald for totals, red for drag
  originalValue: bar.value,
}))
```

For fleet mode (`showFleet && fleetWaterfall`): render a second pair of Bar series with `stackId="fleetStack"`, using blue fill (`#60A5FA` for fleet totals, `#93C5FD` for fleet drags). Both series appear as grouped bars side by side.

**Bar labels:**
